8 Here you'll find the Arduino code used to interface with the SX1509 I2C
9 16 I/O expander. There are functions to take advantage of everything the
10 SX1509 provides - input/output setting, writing pins high/low, reading
11 the input value of pins, LED driver utilities (blink, breath, pwm), and
12 keypad engine utilites.
