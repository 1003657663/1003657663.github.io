Debug ComponentÂ¶

The debug component can be used to debug problems with ESPHome. At startup, it prints a bunch of useful information like reset reason, free heap size, ESPHome version and so on.

# Example configuration entry
debug:

# Logger must be at least debug (default)
logger:
  level: debug

No configuration variables.
