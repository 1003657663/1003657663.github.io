The TMP117 Temperature sensor allows you to use your TMP117 (datasheet, sparkfun) sensors with ESPHome.

The TMP117 is a high precision temperature sensor that communicates over IÂ²C. Each sensor is tested on a NIST tracable test setup during Texas Instrumentsâ€™ production process. Accuracy should be at worst 0.1C across the -20C to +50C temperature range.

IÂ²C AddressesÂ¶

In order to allow multiple sensors to be connected to the same IÂ²C bus, the creators of this sensor hardware have included some options to change the iÂ²c address.

  • If the address pin is pulled to GND, the address is 0x48 (Default).

  • If the address pin is pulled to VCC, the address is 0x49.

  • If the address pin is tied to SDA, the address is 0x4A.

  • If the address pin is tied to SCL, the address is 0x4B.
